# blog-API
A full-featured, RESTful Blog API built with Node.js, Express, MongoDB, and Cloudinary for image storage. Includes JWT authentication, role-based access, email verification, CRUD operations for posts and comments, and Swagger API documentation. Designed for professional, production-ready use and easy frontend integration.


// Features

User authentication & registration with email verification

Role-based access control: admin, author, user

Create, update, delete, and read blog posts

Upload images to Cloudinary

Like/unlike posts

Commenting system with CRUD operations

Admin dashboard for managing users and content

Fully documented Swagger API

Ready for deployment on Render or any cloud provider


//Tech Stack

Backend: Node.js, Express

Database: MongoDB Atlas

Authentication: JWT, bcrypt

File Storage: Cloudinary (for images)

Email Service: SendGrid (for verification)

API Documentation: Swagger/OpenAPI
